# Управление ИТ 
при Центральном Исполнительном Комитете партии "Рассвет"
### Куратор УИТ: Олег Александрович Бедрин

1) Отменить действия в [Инструкции №1](../Инструкция%20№1/Текст.md) и [Инструкции №2](../Инструкция%20№2/Текст.md).
   1) Куратор Проектной группы "Шамир" - освобождается от должности. Её ключ как Куратора Проектной группы "Шамир" - отзывается ([ссылка на PGP файл отзыва](../../pgp/Shamir_Curator_0x78DFCEF8_public.asc))
   2) [Ключ текущего Куратора УИТ отозвать](../../pgp/Directorate_Curator_0xAEDB22DF_public_revoked.asc).
      1) Прилагается [сертификат отзыва](../../pgp/Directorate_Curator_Revocation_Certificate_5314A3B7B115335EDE6222B192542CA8AEDB22DF.rev).
   3) Сгенерировать [новый ключ Куратора УИТ](../../pgp/ITD_74_Curator_0x1504A884_public.asc), которым будут подписываться все официальные документы от имени Куратора УИТ после 03.01.2026.
   4) Все проектные группы созданные при УИТ ЦИК вплоть до 03.01.2026, расформировываются, а репозитории удаляются. Владение исходным кодом и всеми связанными с ним правами регулируется соответствующими лицензиями внутри удаляемых репозиториев в файлах под именем `LICENCE`.
2) Расформировать Управление ИТ при Центральном Исполнительном Комитете партии "Рассвет" по инициативе Куратора УИТ и в связи с выходом Куратора УИТ из ЦИК Партии. (Ниже прикреплен текст заявления о выходе из ЦИК Партии.)
   1) [Письменное заявление о выходе из ЦИК Партии текущего Куратора УИТ](./Выход_из_ЦИК_заявление_Олег_Бедрин.pdf) от 26.12.2025.
3) Создать Управление ИТ регионального отделения по Челябинской области Партии (далее УИТ), на основании новой редакции Регламента - № 3 (от 03.01.2026).
4) Сохранить редакцию Регламента № 2, полностью утратившую свою силу, в качестве [публичного архива](./Регламент%20-%20Редакция%202/Текст.md).
5) Создать и принять в УИТ репозитории связанные связанные с продуктами:
   1) VK мини-апп для партийных рассылок от имени регионального отделения по Челябинской области.
   2) Социологическая платформа "ДАВО "Заря".
6) Учредить проектные группы согласно следующей организации:
   1) В **Отдел серверной разработки**
      1) Проектная группа "Хеллман" - для поддержки серверной части к смарт-контрактам разрабатываемым в группе "Диффи".
      2) Проектная группа "Дейкстра" - для разработки VK мини-аппа для партийных рассылок от имени отделения.
   2) В **Отдел разработки интерфейсов**
      1) Проектная группа "Меркл" - для разработки веб-интерфейсов к смарт-контрактам разрабатываемым в группе "Диффи".
   3) В **Отдел децентрализованных вычислений**
      1) Проектная группа "Диффи" - для объединённой разработки смарт-контрактов в Ethereum для: ААА-процедур, формальной модели "Заря", системы децентрализованного автономного документооборота и прочего.
   4) В **Отдел безопасности**
      1) Проектная группа "Шамир" - для создания инструкций по созданию, обмену и хранению секретов отделения Партии (приватных ключей, доступов и проч.)
   5) В **Отдел кадров**
      1) Проектная группа "Керниган" - для разработки процессуальной документации (регулирование процесса контрибьютинга в репозитории УИТ ЦИК), а также для работы с контрибьюторами (из публики) и мейнтейнерами (из числа сопартийцев-сотрудников Управления ИТ) репозиториев УИТ.
